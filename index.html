上线指南
让全世界都能访问你的网站
按以下步骤操作，5分钟完成部署，完全免费！
1

🏷️ 重命名入口文件 ○

Vercel 识别 index.html 为网站入口。将当前的 index.html 保持不变，这是正确的。

2

📦 准备 GitHub ○

如果还没有 GitHub 账号，先去注册一个：github.com（用邮箱注册，完全免费）

💡 GitHub 是代码仓库，用来存放你的网站代码
3

⬆️ 上传到 GitHub ○

在 GitHub 新建一个仓库（Repository），命名为 chuang-ai，设置为 Public（公开），然后把 index.html 上传进去。

💡 可以用 GitHub 网页直接上传：点击仓库 → Add file → Upload files → 拖入 index.html → Commit changes
4

🚀 连接 Vercel ○

打开 vercel.com，用 GitHub 账号登录，点击 "Add New Project"，选择刚才的 chuang-ai 仓库，点击 Deploy！

💡 Vercel 会自动检测到 index.html 并配置好一切，无需任何设置
5

🌐 获得你的网址 ○

部署完成后，Vercel 会给你一个免费域名，格式类似：chuang-ai.vercel.app

任何人都可以通过这个网址访问你的AI创作平台！

💡 可选：用阿里云/腾讯云购买域名（约50元/年），绑定到 Vercel，获得你自己的品牌域名

🎉 恭喜！你的AI平台已上线！

任何人都可以通过这个网址访问，所有功能（AI绘图、视频、积分系统）全部正常工作！
